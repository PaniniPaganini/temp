# 🍗 Happy Emoji: A Vibe-Coding Experiment

A rapid-prototype emoji matching game featuring a "Fight the Landlord" (欢乐斗地主) aesthetic.

---

## 🕹️ How to Play

This game is a **single-file web application**. No installation, no `npm install`, and no complex setup required.

### **Option 1: Desktop (PC/Mac)**
1. **Download** the `Happy Emoji.html` file from this repository.
2. **Double-click** the file to open it in your default browser (Chrome/Edge/Safari). 
3. Enjoy the vibes!

### **Option 2: Mobile (iOS/Android)**
1. **Open** the file using any browser app on your phone.

*Note: Some browsers may auto-block background music (BGM).*

---

## 🛠️ Technical Highlights

* **Architecture:** Single-file HTML5 / CSS3 / Vanilla JavaScript.
* **Development Method:** **Vibe Coding** (LLM-assisted rapid iteration).
* **Key Features:** Responsive grid layout optimized for both desktop and mobile.
    * Custom state management for emoji matching and scoring logic.
    * Nostalgic BGM integration (The "Fight the Landlord" classic).
